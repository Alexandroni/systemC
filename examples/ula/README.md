Arithmetic Logic Unit
----------------------------------------------------------------------------------------

This is a example of a ALU (Arithmetic Logic Unit) with 3 operations implement using SystemC<br/>

-- Vivado HLS 2017.3<br/>
----- SystemC 2.1<br/>
-- Vivado Design 2017.3<br/>
----- VHDL<br/>

Inputs: <br/>
A - number<br/>
B - number <br/>
op - operator <br/>

op = 0 -> output to zero <br/>
op = 1 -> Add (A+B)   <br/>
op = 2 -> Sub (A-B) <br/>
op = 3 -> Mult (A*B) <br/>
